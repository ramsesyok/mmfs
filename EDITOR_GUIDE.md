# Revideo エディターの使い方

## エディターへのアクセス

エディターは **http://localhost:9000** で起動しています。

ブラウザで開くと、Revideoのビジュアルエディターが表示されます。

## 動画のプレビュー

1. エディター画面の下部にあるタイムラインで再生ボタン（▶️）をクリック
2. 動画が再生され、以下の要素が表示されます：
   - スライド背景（1440×810）
   - 字幕（緑の枠線、1360×130）
   - キャラクター（赤の枠線、460×950）
3. 各シーンは3秒間再生されます（合計6秒）

## 動画のレンダリング（書き出し）

エディター内で動画を書き出すには：

1. 画面右上の **Render** ボタンをクリック
2. レンダリング設定を確認：
   - 解像度: 1920×1080
   - フレームレート: 60fps
3. **Start Rendering** をクリック
4. レンダリングが完了すると、動画ファイルがダウンロードされます

## カスタマイズ

動画をカスタマイズするには、`public/config.json` を編集してください：

```json
{
  "scenes": [
    {
      "slidePath": "your_slide.png",
      "voicePath": "your_voice.wav",
      "text": "あなたの字幕テキスト"
    }
  ]
}
```

ファイルを保存すると、エディターが自動的にリロードされます。

## トラブルシューティング

### エディターが表示されない場合

```bash
# サーバーを再起動
npm run editor
```

### ポート9000が使用中の場合

```bash
# 別のポートで起動
npx revideo editor --port 9001
```

## 現在の制限事項

- 音声の再生時間は固定（各シーン3秒）
- 動的な音声長に対応するには、Revideoの`Audio`コンポーネントを使用する必要があります

## 次のステップ

1. エディターで動画をプレビュー
2. レイアウトと位置を確認
3. 必要に応じて`config.json`を編集
4. 動画をレンダリング
